flextable

flextable 是一个用于报告和发表提供创建表格的包。

library(flextable)

flextable()可以从数据框data.frame创建报告表格。只需要在其中填入数据框作为参数即可实现。

ft <- flextable(airquality[ sample.int(10),])
ft
  • set_caption：设置标题；
  • color：设置字体颜色，其中参数part可以对表格的不同部分进行设置；
  • add_footer_lines：添加脚注；
  • add_header_row：添加首行，同时设置参数长宽colwidths；
ft <- add_header_row(ft,
  colwidths = c(4, 2),
  values = c("Air quality", "Time")
)
ft <- theme_vanilla(ft)
ft <- add_footer_lines(ft, "Daily air quality measurements in New York, May to September 1973.")
ft <- color(ft, part = "footer", color = "#666666")
ft <- set_caption(ft, caption = "New York Air Quality Measurements")
ft

其中使用了主题theme_vanilla来作为整体的主题，其他的主题还有包括get_flextable_defaults()来查看现有的默认主题的参数设置。

恢复默认主题可使用init_flextable_defaults()实现：

init_flextable_defaults()

更多细节

flextable对表格有更多的细节上的设置：

myft <- flextable(head(mtcars), 
  col_keys = c("am", "carb", "gear", "mpg", "drat" ))
myft
myft <- italic(myft, j = 3)
myft <- color(myft, ~ drat > 3.5, ~ drat, color = "red")
myft <- bold(myft, ~ drat > 3.5, ~ drat, bold = TRUE)
myft

这里的color对drat属性满足一定条件下则设置其他颜色；同样若drat满足条件下则加粗。

设置字体

dat <- data.frame(
  wool = c("A", "B"),
  L = c(44.56, 28.22), 
  M = c(24, 28.77), 
  H = c(24.56, 18.78)
)
flextable(dat) |> 
  font(part = "all", fontname = "Inconsolata")
set_flextable_defaults(
  font.family="Inconsolata", 
  cs.family = "Symbol", 
  eastasia.family = "仿宋")
df = data.frame(
  `标题Title`='微软MS',
  `内容Content`='无题lorem',
  stringsAsFactors = FALSE)
ft = df %>% 
  flextable %>% 
  autofit %>% 
  width(width=1) %>% 
  theme_box()
ft

一个flextable是由三部分组成：

  • header: 默认下只有一个header行包括数据框行名；
  • body: body部分是包含数据框的数据；
  • footer: footer没有默认，但可以包括任何内容。

ft <- flextable(head(airquality))
ft

Selectors

选择器（Selectors）可用于某行某列对其进行操作。很多的flextable函数有选择器i和j包括：bg(),bold(),hline(),color(),align()等。

# data prep ----
library(DT)
library(data.table)
## 
## Attaching package: 'data.table'
## The following objects are masked from 'package:dplyr':
## 
##     between, first, last
dat <- head(ggplot2::diamonds, n = 10)
qflextable(dat) %>% 
  color(~ price < 330, color = "orange", ~ price + x + y + z ) %>% 
  color(~ carat > .24, ~ cut, color = "red")

默认下的i和j为空值（NULL），NULL可认为是所有的行和所有的列都可实现。

i用于行，j用于列的调整；

使用i = ~ col %in% "xxx"选择所有的行中的列col是xxx的。其中的变量表达式为~。若需要使用多重条件，可以使用&和|来实现。

  • 运算符+可用于多重列。
ft <- qflextable(dat)

color(
  ft, i = ~ cut %in% "Premium", 
  j = ~ x + y, color = "red")

变量j支持使用所有的col_key组成的向量进行操作。

ft %>% 
  color(j = c("x", "y"), color = "orange", part = "body") %>% 
  bold(j = c("price", "x"), bold = TRUE,part = "all")

同样也通过一个col_key数字实现：

color(ft, i = 2:3, j = 2:5, color = "red")

逻辑运算

color(ft, i = rep(c(TRUE, FALSE), 5), color = "blue")

背景设置

bg(ft, i = ~ price < 335, 
  j = c('x', 'y', 'z'), 
  bg = "orange", part = "body")

边框

library(officer)
big_border = fp_border(color="red", width = 2)
small_border = fp_border(color="gray", width = 1)

myft <- flextable(head(airquality))
myft <- border_remove(x = myft)
myft <- border_outer(myft, part="all", border = big_border )
myft <- border_inner_h(myft, part="all", border = small_border )
myft <- border_inner_v(myft, part="all", border = small_border )
myft

输出

flextable支持多种输出格式，包括html、docx、ppt、pdf。在Rstudio中默认为html的输出格式。

  • save_as_html()
  • save_as_pptx()
  • save_as_docx()
  • save_as_image()

若需要进行提前预览，可使用print(ft,previed=)实现。

print(ft, preview = "docx")

仍然有更多的内容可以得到探索。 可参考flextable的官方指南 https://ardata-fr.github.io/flextable-book/index.html
